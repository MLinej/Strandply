1. Create a new Google Sheet → Extensions → Apps Script

3. Set Execute as: Me, Who has access: Anyone → Deploy → copy URL above
4. Spreadsheet ID is the long string in your Sheet URL: docs.google.com/spreadsheets/d/SHEET_ID/edit
5. Click Save Config then Push to Sheets — data syncs automatically
